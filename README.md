

# Customer Behavior Analysis & Customer Segmentation

## Project Overview

This project analyzes customer transaction and behavioral data to identify customer segments, purchasing patterns, and churn risks. Using data analytics and machine learning techniques, the project helps businesses understand customer behavior, improve retention strategies, and make data-driven marketing decisions.

## Objectives

* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Engineer customer behavior features
* Segment customers using RFM (Recency, Frequency, Monetary) analysis
* Apply K-Means clustering for customer grouping
* Analyze purchase patterns and customer retention trends
* Identify customers at risk of churn
* Generate actionable business recommendations
